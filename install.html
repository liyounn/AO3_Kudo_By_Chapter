♥ Chapter Kudos
Bookmarklet Install Guide

The bookmarklet adds chapter kudos buttons to any AO3 work that has the skin applied. It works on any browser, any device — including mobile.

♥ AO3 Chapter Kudos
On desktop: drag this to your bookmarks bar ↑

Install by platform

1
Show your bookmarks bar if it's hidden.
Chrome: Ctrl+Shift+B  |  Firefox: View → Toolbars → Bookmarks
2
Drag the red button above into your bookmarks bar. That's it — you're done.
3
To use it: visit an AO3 work with the Chapter Kudos skin, then click the bookmark. Kudos buttons will appear at the end of each chapter.
